authorize

authorize(credentials: Object, callback: function)
Parameters

callback (function) The callback to call with the authorized client.

getNewToken

Get and store new token after prompting for user authorization, and then execute the given callback with the authorized OAuth2 client.

getNewToken(oauth2Client: google.auth.OAuth2, callback: getEventsCallback)
Parameters
oauth2Client (google.auth.OAuth2) The OAuth2 client to get token for.
callback (getEventsCallback) The callback to call with the authorized client.
